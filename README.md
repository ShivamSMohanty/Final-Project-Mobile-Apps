# Final-Project-Mobile-Apps
Shivam, Roshan, and Vedant's Final Project for Mobile Apps

The goal of this application is to create a personalized anime and show tracking platform that helps users organize what they have watched, discover new content, and analyze their viewing habits. The app aims to simplify tracking while also enhancing user engagement through recommendations and statistics.

Key Features Include: 
A personalized watchlist to track completed, ongoing, and planned shows
A recommendation system that suggests new anime based on user preferences using swipe interactions or viewing history
Integration with third-party APIs (AniList and Jikan) to fetch anime data, ratings, and user history
User profiles that store individual watch history, preferences, and statistics
A statistics dashboard showing insights such as total episodes watched, most-watched genres, and fastest completed series
A swipe-based interface for discovering and rating new shows


Project Architecture:
The app will use a multi-screen mobile frontend (login, recommendations, watchlist, stats) connected to third-party APIs like AniList and Jikan for anime data and user history. User-specific data will be stored locally or in a simple backend, allowing each profile to maintain its own watch history, preferences, and statistics.
